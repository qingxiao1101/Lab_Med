com + /  注释
Command + T：快速显示当前类的继承结构
Option + ↓：当前行和下面一行交互位置
Option + ↑：当前行和上面一行交互位置
Command + Shift + X：把当前选中的文本全部变为大写
Command + Shift + Y：把当前选中的文本全部变为小写
Command + Shift + P：定位到对于的匹配符（譬如{}）


com+ [ ] 窗口切换


【com+O】 显示类中方法和属性的大纲，能快速定位类的方法和属性
Ctrl+K】、【Ctrl++Shift+K】 快速向下和向上查找选定的内容
Ctrl+Shift+G】 查找类、方法和属性的引用。这是一个非常实用的快捷键，例如要修改引用某个方法的代码，可以通过【Ctrl+Shift+G】快捷键迅速定位所有引用此方法的位置。
